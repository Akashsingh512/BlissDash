This app is used by volunteers to collect & track YES+ / AOL lead data.

### What it does

- add a new lead with name / phone / comment / location / who met
- auto-timestamp every lead on submit
- save default values (date + location + media link)
- store all data directly into Google Sheets (1 single master sheet)
- show last 10 leads submitted in UI
- open call / whatsapp directly from list
- edit an existing lead (follow-ups & status)
- update rows back into sheets
- store whatsapp template and reuse it

### where it runs

- frontend = React (Cloudflare Pages)
- backend = Cloudflare Function
- database = Google Sheet

### who this is for

volunteers / core team who want a **fast & painless** way to log leads without Google Sheets manual work.
